# 📝 Project Description

CodeBuddy is a beginner-friendly coding challenge generator developed using Python fundamentals.

The program generates random programming challenges according to the selected programming language and difficulty level. It also provides helpful hints, learning topics, estimated completion time, achievement badges, and motivational quotes to encourage continuous learning.
